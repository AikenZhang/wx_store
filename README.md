# fashion衣妆(小程序)
## 功能
- [x] 商品列表
- [x] 类别尺寸选择
- [x] 分页
- [x] 分类
- [x] 排序
- [x] 购物车
- [x] 收藏
- [x] 收货地址
- [x] 登录/登出
- [x] 订单
## 组件 
- [x] 数值选择 （counter）
- [x] 列表 （fyCell）
- [x] 加载更多 （fyLoadMore）
- [x] 价格 （fyPrice）
- [x] 排序 （fyTab）
- [x] 左滑删除 （leftSlide）
- [x] 分页 （paging）
